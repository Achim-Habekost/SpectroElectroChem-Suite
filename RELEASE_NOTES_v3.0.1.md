# SpectroElectroChem Suite v3.0.1

First public release of SpectroElectroChem Suite.

## Features

- Raman spectrum analysis
- Raman and SERS voltammograms
- Absorptovoltammograms
- Fluorovoltammograms
- Interactive 3D surface plots
- Interactive waterfall plots
- Baseline correction
- Savitzky–Golay smoothing
- Excel export
- PDF export
- Plugin architecture

## Included modules

1. Raman Spectrum Analysis
2. SERS / Raman Voltammogram
3. Absorption / Fluorescence Voltammogram

## Release notes

This release provides the first public version of the source code, documentation, installation scripts and packaging templates. The Raman/SERS voltammogram module includes control of the waterfall vertical offset and exports waterfall data to Excel.
