SpectroElectroChem Suite v4.0.0

Please do the following:

1. Unzip this folder completely.
2. Double-click:
   Install_required_Python_packages.bat
   This only has to be done once.
3. Double-click:
   Test_Installation.bat
4. Double-click:
   Start_SpectroElectroChem_Suite.bat

Important:
Do not start files while they are still inside the ZIP archive.
The folder must be extracted first.

For scientific citation, please use the GitHub release and the Zenodo DOI after archiving.
